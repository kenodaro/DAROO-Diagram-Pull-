# DΛRØØ DiagramPull

A browser-based tool for extracting, cropping, and exporting diagrams from PDF documents

## 🚀 Features
- Upload PDF files
- Extract diagrams automatically
- Crop manually with precision
- Queue multiple selections
- Export all crops as a ZIP

## 🧠 Use Case
Built for educators and students working with exam papers who need fast access to diagrams without manual screenshotting.

## ▶️ How to Use
1. Open `index.html`
2. Upload your PDF
3. Crop the diagrams you need
4. Export all crops in one click
